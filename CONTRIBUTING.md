# Contributing to Help Desk

Merci de vouloir contribuer à ce projet !

## Workflow de Développement

1.  Forker le projet.
2.  Créer une branche pour votre fonctionnalité (`git checkout -b feature/ma-feature`).
3.  Commiter vos changements (`git commit -m 'Add some feature'`).
4.  Pusher vers la branche (`git push origin feature/ma-feature`).
5.  Ouvrir une Pull Request.

## Standards

-   **Backend**: Python (FastAPI), respect de PEP8, tests avec Pytest.
-   **Frontend**: React, ESLint.
-   **Commits**: Conventional Commits (ex: `feat: add login page`).

## Lancer le projet

```bash
docker-compose up --build
```
